# 2026-03 鸿蒙前沿追踪与落地记录

## 生态信号

### HarmonyOS 6 (API 20 / 22)
- **API 20** (6.0.0) 已发布开发者版本（2025-09-25）
- **API 22** (6.0.2) 公测进行中
- ArkUI 新特性：Text 文字色渐变、中西文自动间距、SymbolGlyph 阴影/渐变、Swiper 滑动状态回调、数字翻转动画
- `@Consume` 装饰器支持设置默认值
- 文件系统沙盒安全边界加强，`copy` 接口行为变更
- `router` 模块持续被 Navigation + NavPathStack 取代

### 仓颉语言
- LTS 1.0.0 已发布并开源（编译器 + 运行时 + 标准库）
- 与 ArkTS 并行定位，暂不纳入本插件迭代
- 后续视社区采纳率评估是否加入语法支持

### 社区高频问题
- 搭骨架重复代码多（列表页、详情页、登录页）
- 从 Android 迁移后文件系统读写失败（沙盒路径变化）
- 不知道 `router` 已废弃，仍在使用旧 API
- 第三方库少，OHPM 生态待完善

### 诊断 (src/language/)
- **DEPRECATED_ROUTER**: 检测 `import router from '@ohos.router'` 和 `router.pushUrl()` 等调用，建议迁移到 Navigation + NavPathStack
- **SANDBOX_HARDCODED_PATH**: 检测硬编码 `/data/` 和 `/storage/` 路径，建议使用 `getContext().filesDir` 等沙盒 API
- 两条规则均配套 Quick Fix（打开官方文档）
